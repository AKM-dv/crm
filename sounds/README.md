# Notification Sound Files

Place your audio files in this directory:

1. **general.wav** - For general notifications:
   - New lead assigned
   - Lead reassigned
   - Follow-up reminders (upcoming)
   - Task assigned
   - Task reassigned
   - Admin notifications

2. **important.wav** - For important/urgent notifications:
   - Overdue follow-ups
   - Missed follow-ups
   - Pending follow-ups
   - Tasks due soon
   - Overdue tasks

3. **converted.wav** - For lead conversion:
   - Lead status changed to "won"

## File Requirements:
- Format: WAV (recommended) or MP3
- Location: `/frontend/public/sounds/`
- The system will automatically use these files based on notification type
